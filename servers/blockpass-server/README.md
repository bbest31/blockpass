
---
## 1.Install
### MongoDB
> Download [MongoDB](https://www.mongodb.com/try/download/community) and make sure an instance is running.

### npm
```
npm i
```

## 2.Start
### Run as dev using nodemon
```sh
npm run dev
```

---
## Dependencies
- express v4.18.1
- mongoose v6.3.4
